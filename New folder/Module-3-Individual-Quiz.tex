\item
  Brand A gasoline was used in 16 similar automobiles under identical
  conditions. The corresponding sample of 16 values (miles per gallon)
  had mean 19.6 and standard deviation 0.4. Under the same conditions,
  high-power brand B gasoline gave a sample of 16 values with mean 20.2
  and standard deviation 0.6. Is the mileage of B significantly better
  than that of A? Assume normality. Test the hypothesis using both
  P-value and fixed significance level with Î±=0.05 approaches (if
  possible).
\end{enumerate}

Where:

n = 16,

\(\mu_{a}\) = 19.6

\(\sigma_{a}\) = 0.4

\(\mu_{b}\) = 20.2

\(\sigma_{b}\) = 0.6

\hypertarget{section-1}{%
\subsubsection{}\label{section-1}}

\(H_{0}\) = \(\mu_{a}\) - \(\mu_{b}\) \textless{} 0

\(H_{1}\) = \(\mu_{a}\) - \(\mu_{b}\) \textgreater{} 0

\hypertarget{section-2}{%
\subsubsection{}\label{section-2}}

\(\sigma^2_{a}\) = \(0.4^2\)

\(s_{a}\) = 0.16

\(\sigma^2_{b}\) = \(0.6^2\)

\(s_{b}\) = 0.36

\hypertarget{section-3}{%
\subsubsection{}\label{section-3}}
